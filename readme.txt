Elementor Addon Pack For 99fy - Free WooCommerce Theme

== Description ==
99Fy Core is a plugin for 99fy - Free WooCommerce Theme. This plugin requires the [99fy](https://wordpress.org/themes/99fy/) theme to be installed.

== What is 99fy? ==
99fy is a Free WooCommerce WordPress Theme, it is responsive and fast loading theme. 99 Home pages are included in this theme. 99fy using [Elementor Page Builder](https://wordpress.org/plugins/elementor/) to create pages easily,
and supports popular WooCommerce Addon - WooLentor. [WooLentor](https://wordpress.org/plugins/woolentor-addons/) helps to create custom templates for shop and product pages. All of the popular niches for WooCommerce Stores are included in this theme. More niches are added regularly.

== Installation ==
This section describes how to install the 99Fy Core.

= 1) Install =

1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "99Fy Core".
3. Install, then Activate it.

= Or: =
1. Unzip (if it is zipped) and Upload `99fy-core` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
